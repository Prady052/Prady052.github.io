RagBase — self-hosted RAG platform

In active development
Status: shipping core pipeline
Enterprise RAG

Enterprises need to query thousands of internal documents without shipping sensitive data to a third-party AI API, and keyword search can't reason about meaning. RagBase is a 100% self-hosted retrieval-augmented generation platform that lets teams upload documents and ask questions in plain language, with every model call running on local infrastructure.

→
Chunk + embed (nomic-embed-text)

Qdrant vector search
→
llama3.2:3b (Ollama)
→
Cited answer via SSE
<60sPROCESSING · 50-PAGE DOC
<8sQUERY RESPONSE LATENCY
$0COST PER QUERY, ZERO CLOUD CALLS

Backend is FastAPI with JWT-based role access control; the React frontend supports multi-turn chat persisted in PostgreSQL with real-time streaming answers. The whole stack — API, workers, vector DB, and LLM runtime — is containerized with Docker Compose for offline-first deployment on local infrastructure.
